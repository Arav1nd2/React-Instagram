# React-Insta-Clone

## A simple Instagram Clone made using React,Redux and Firebase

### Features : 
- Login/Logout (Firebase Authentication)
- Facebook Login
- Add likes to an existing post
- Working Search Bar (Filters the post with Username)
- Add Comments to any post
